# Prompt Engineering Task - Neurofive Solutions

## Overview
This repository demonstrates the comparative efficacy of **Chain-of-Thought (CoT + Persona Prompting)** versus **Plain Prompting** for financial revenue forecasting.

## Prompt Comparison

### 1. Plain Prompt
* **Output:** Basic calculations, raw text, lacking structural depth and executive context.

### 2. CoT + Persona Prompt
* **Prompt Used:** 
  > "You are a senior financial analyst and data expert at Neurofive Solutions. Think step-by-step before answering, showing all calculations clearly..."
* **Output:** Boardroom-ready executive report featuring step-by-step financial decay models and key stakeholder assumptions.
